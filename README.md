# 🚀 GUNTANK - 2D Roguelite Survival

**GUNTANK** is a 2D survival game (Roguelite Survival) developed using Python and the Pygame library. Inspired by Vampire Survivors and the classic Battle City, players take control of a lone tank to fight against endless waves of enemies, collecting resources to upgrade their power both during and between runs.

---

## 🎮 Key Features
- **Auto-aim & Auto-skill:** The system automatically targets the nearest enemy and activates special skills periodically. You only need to focus on moving (Hit and Run).
- **Diverse Tank System:** 4 distinct types of tanks with unique combat styles (Missiles, AoE Fire Ring, Piercing Laser).
- **Roguelite In-game Upgrades:** Level up during the battle to receive 3 random skill choices (Max HP Up, Lifesteal, Damage Up, Piercing Bullets, etc.).
- **Meta-progression:** Keep the Gold collected after each run to permanently upgrade base stats and unlock new tanks in the preparation lobby.
- **Dynamic Difficulty:** Enemies become more numerous, tankier, and deal more damage based on how long you survive.

---

## 🛠️ Technologies Used
- **Language:** Python 3.x
- **Main Library:** `pygame` (Graphics, Audio, & Event handling)
- **Supporting Libraries:** `math`, `random` (Trigonometry calculations and randomization systems)
